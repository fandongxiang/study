js是web浏览器“执行”的“指令”

语句由值、运算符、表达式（由变量、运算符、值构成）、关键词（告诉浏览器执行的动作）、注释构成

js代码块

用｛...｝括起来，作用是定义一系列按顺序同时执行的代码

JavaScrip中的关键词

关键词 描述
break 终止switch或循环
continue 终止循环并在顶端开始
debugger 停止运行JavaScript，并运行调试函数（如果可用）
do...while 执行语句块，并在条件为真时循环执行代码块
for 标记需执行的语句块，只要条件为真
function 申明函数
if...else 标记需执行的语句块，满足条件时
return 退出函数
switch 标记需执行的条件块，根据不同情况
try...catch 对语句块进行错误处理
var 申明变量
