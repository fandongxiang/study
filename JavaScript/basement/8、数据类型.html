数据类型分类

  1. 数字:var a = 1;
  2. 字符串:var LastName = "fan";
  3. 数组:var car = ["porsche","BMW"."VOLVO"]
  4. 对象:var x = {firstname:"fan".lastname:"dognxiang"}

原始数据

没有额外属性和方法的单一简单数据值 typeof返回的原始数据类型有4种
  • string
  • number
  • boolean
  • undefined

复杂数据类型

typeof对复杂数据会返回两种类型
  1. function:仅限于函数
  2. object:对象、数据或null
