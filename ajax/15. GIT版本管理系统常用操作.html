git中的三种状态

已修改

已暂存

已提交

配置用户信息

配置用户信息：git config --global user.name '用户名'

配置用户邮箱：git config --global user.email '邮箱'

查看全局配置信息：git config --list --global

获取帮助信息

网页获取帮助信息：git help config

本地获取帮助信息：git config -h

创建git仓库

在当前目录创建git仓库：git init (当前目录生成一个.git的隐藏目录)

git工作区的四种状态

未跟踪：untracked 未被git管理的文件

未修改：unmodified 工作区中的文件和git仓库中的一致

已修改：modified 工作区中的文件和git仓库中的不一致

已暂存：staged 工作区中的被修改的文件被放到暂存区，但未被放到git仓库

检查文件的状态

检查完整状态：git status

检查精简状态：git status -s

跟踪文件

跟踪单个文件：git add 文件名

跟踪当前目录所有文件：git add .
