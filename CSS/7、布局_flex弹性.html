flex弹性布局

  1. flex实际用途
    1. 用于网站的弹性布局
    2. 常用网页的各种导航栏的布局
  2. flex优点
    1. 可根据内部内容弹性伸缩
  3. flex缺点
  4. flex用法
    1. 设置一个父元素（容器）
    2. 在父元素容器中定义子元素（项目）
  5. 父元素（容器）属性
    1. display:flex 定义父元素为可伸缩flex容器
    2. flex-direction 定义堆叠方向
      • column 定义堆叠方向从上到下
      • column-reverse 定义堆叠方向从下到上
      • row 定义堆叠方向从左到右
      • row-reverse 定义堆叠方向从右到左
    3. flex-wrap 定义是否应该对flex项目换行
    4. flex-flow 同时设置flex-direction和flex-wrap简写属性，例如row wrap
    5. justify-content 定义flex项目的水平对齐方式
      • center 子项目在父容器中心对齐
      • flex-start 子项目在父容器开头对齐
      • flex-end 子项目在父容器末端对齐
      • space-around 子项目之前、之后、之间自动用空格填充
      • space-between 子项目之间用空格填充
    6. align-items 定义flex单排子项目的垂直对齐方式
      • stretch 拉伸flex项目以填充容器（默认）
      • center 子项目在父容器中心对齐（用于子项目高度一定时）
      • flex-start 子项目在父容器顶部对齐（用于子项目高度一定时）
      • flex-end 子项目在父容器底端对齐（用于子项目高度一定时）
      • baseline 子项目沿父容器基准线对弈（用于子项目高度不同时）
    7. align-content 定义多排子项目弹性线的垂直对齐方式（与wrap共用）
      • stretch 拉伸弹性线以占满空间（默认）
      • space-between 定义弹性线之间有相等的间距
      • space-around 定义弹性线之前、之间、之后有相等的间距
      • center 子项目在容器中间显示
      • flex-start 子项目在容器开头对齐
      • flex-end 子项目在容器末端对齐
  6. 子元素项目属性
    1. order 定义子项目的显示顺序，默认为0 html中
    2. flex-grow 定义子项目相比于其它子项目的拉伸宽度，默认为0
    3. flex-shirink 定义子项目相比于其它子项目的缩减宽度，默认为1
    4. flex-basis 定义子项目的初始长度
    5. flex 定义flex-grow、flex-shirink、flex-basis的简写
    6. align-self 定义子项目的单独对齐方式（center、flex-start、flex-end），将覆盖父容器的对齐方式设置
