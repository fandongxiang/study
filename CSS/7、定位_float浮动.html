CSS浮动

作用

1、实现文字环绕；
2、实现横向布局

缺点

1、布局时缺乏弹性，元素内容会溢出；
2、需要使用clear消除浮动；
  1. float属性
    1. left:向左浮动
    2. right:向右浮动
    3. none:元素不浮动
    4. inherit:从父级继承浮动属性
  2. float浮动的特点
    1. 不浮动元素会占据浮动元素空间(脱离了文档流)，但元素内文字会环绕浮动元素分布
    2. 浮动找浮动
    3. 多个浮动元素，以顶部基准对齐
    4. 浮动后的元素会变成行内块元素，可以定义宽高等属性
  3. clear属性：去掉浮动属性，包括从父级继承来的属性
    1. left、right：去掉向左向右浮动属性
    2. both:去掉所有浮动属性
    3. inherit:去掉父级继承来的属性
